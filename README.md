# Bottle - simple game engine written on C++

## Bottle features:
 - Vulkan renderer
 - ECS architecture (but CPB)

## CPB Systems
 - Graphics
 - Event
 - Animation (planned)
 - Input/Output (planned)
 - Window

## Current status:
⚠️ Early prototype, not functional
